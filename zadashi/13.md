1. Стек (Stack)
Определение
Стек - это структура данных работающая по принципу LIFO последним пришёл, первым ушёл.
Последний добавленный элемент удаляется первым.

Основные операции
push(x) - Добавляет элемент x на вершину стека
pop - Удаляет элемент с вершины стека и возвращает его
peek / top - Возвращает элемент на вершине без удаления
isEmpty - Проверяет, пуст ли стек
size - Возвращает количество элементов
    push(3)      push(5)      pop() -> 5
       ↓            ↓            ↓
+---+  +---+  +---+  +---+  +---+  +---+
|   |  | 3 |  |   |  | 5 |  |   |  | 3 |
+---+  +---+  +---+  +---+  +---+  +---+
|   |  |   |  | 3 |  | 3 |  | 3 |  |   |
+---+  +---+  +---+  +---+  +---+  +---+
Аналогии из реальной жизни
Стопка тарелок: последнюю положенную тарелку берут первой
Кнопка "Назад" в браузере: последняя посещённая страница открывается первой при возврате
Отмена действий: последнее действие отменяется первым
Рюкзак: чтобы достать вещь, положенную первой, нужно вытащить все сверху

2. Очередь (Queue)
Определение
Очередь - это структура данных работающая по принципу FIFO первым пришёл, первым ушёл.
Первый добавленный элемент удаляется первым.

Основные операции
Операция - Описание
enqueue - Добавляет элемент x в конец очереди
dequeue - Удаляет элемент из начала очереди и возвращает его
front - Возвращает первый элемент без удаления
isEmpty - Проверяет, пуста ли очередь
size - Возвращает количество элементов
enqueue(1)  enqueue(2)  enqueue(3)  dequeue() -> 1
    ↓           ↓           ↓             ↓
[ 1 ] → [ 1, 2 ] → [ 1, 2, 3 ] → [ 2, 3 ]
front = 1           front = 1    front = 2
rear  = 1           rear  = 2    rear  = 3

Аналогии из реальной жизни
Очередь в магазине: первый покупатель обслуживается первым
Очередь печати: документы печатаются в порядке отправки
колл-центр: звонки обрабатываются в порядке поступления
Конвейер: детали обрабатываются в порядке поступления
